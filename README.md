### üîí Security Scanning

This project uses GitHub Actions to automatically scan dependencies for vulnerabilities using:

- **CodeQL** (built-in)
- **Snyk** (requires `SNYK_TOKEN` secret)

To enable Snyk locally:

```bash
npm install -g snyk
snyk auth <your_token>
snyk test
```
